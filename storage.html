Storage

Storage is the long-term memory of the computer. Nowadays there are two types of storage used, Hard Drives (HDDs) and Solid-State Drives (SSDs).

Hard Drives

A hard drive is a form of magnetic mechanical storage, used to store large files such as the operating system, applications and any other files the user needs to store. Now slowly being replaced by Solid-State Drives because of their improved speed, the hard drive is still used alongside solid-state drives for large files because of lower cost for larger sizes, as well as slightly better reliability.

Solid-State Drives

In recent times, it has become increasingly popular to use Solid-State Drives in collaboration with traditional hard drives, the solid state drive containing a users operating system and frequently used applications (such as a web browser) and hard drives are used to store big files (such as video games or videos and pictures.)
