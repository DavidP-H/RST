Memory

RAM (or Random-Access-Memory) is a short-term memory that the computer uses when running programs, a good example is tabs in a web browser, even though you are not using the tabs when you click on them they appear almost instantly, this is because RAM is a very fast form of storage and any information stored in RAM can be retrieved very quickly. Not all RAM is created equal, different units can have different rated speeds (measured in megahertz) different first-word latency and different CAS latency (measured in clock cycles). There are also different generations of RAM, with the most recent being DDR4 (Double Data Rate 4 Synchronous Dynamic Random-Access Memory).
