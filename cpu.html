Processors

You will often hear processors referred to as CPUs CPU stands for Central Processing Unit. The CPU of a computer is sometimes called the brain of a computer, this has some reasoning behind it, but isn't a very descriptive term. An average modern Central Processing Unit in a computer has around 2-8 processing cores, these cores are very powerful because they can do many different tasks and be programmed to do almost anything. This is where the brain comparison comes from.
